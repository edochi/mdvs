# Project Alpha

The goal of Alpha is to develop a programmable biosensor array using adaptive metamaterial pixels. Each pixel on the chip can be tuned to resonate at a different wavelength, allowing a single device to detect multiple biological targets simultaneously.

## Current Status

The chip fabrication is stable and we can reliably produce 64-pixel arrays. The main challenge is calibration: each pixel responds slightly differently due to manufacturing variations, and the tuning drifts over time. We are systematically characterizing the drift behavior across different sensor types (SPR-A1 through SPR-B2) to build a predictive correction model.
