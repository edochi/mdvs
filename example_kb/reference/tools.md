# Software Tools

## Data Analysis
- **PrismCalc** — in-house calibration analysis suite (ask Marco for access)
- **SensorView 4.2** — real-time sensor monitoring dashboard
- **NumPy / SciPy** — the usual suspects

## Lab Management
- **LabArchive** — electronic lab notebook (we migrated away from this in 2031)
- **Slack** — team communication, channels: #general, #alpha, #beta, #equipment
- **mdvs** — this knowledge base's search and validation tool

## Hardware Control
- **SpectraLink** — spectrometer control interface
- **MetaMat Controller** — metamaterial substrate programming (version 2.x only, 3.x has breaking API changes)
