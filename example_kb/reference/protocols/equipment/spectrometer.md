# Spectrometer Operating Protocol

## Startup

1. Power on the main unit (rear switch) and wait for the self-test to complete (~90 seconds).
2. Launch SpectraLink on the control workstation.
3. Verify the connection indicator shows green.
4. Run the daily calibration routine: SpectraLink → Calibrate → Standard Reference.

## Operation

- Always use the appropriate cuvette holder for your sample type.
- Wavelength range: 200–1100 nm. Do not attempt measurements outside this range.
- Integration time: start with 100ms, adjust based on signal strength.
- For metamaterial substrate measurements, use the custom holder (drawer 3, labeled "META").

## Shutdown

1. Remove all samples and clean the cuvette holder.
2. Run SpectraLink → Maintenance → Lamp Hours to log usage.
3. Power off via SpectraLink first, then the rear switch.

## Maintenance

Lamp replacement every 2000 hours. Contact Sara to schedule. Do not attempt to replace the lamp yourself — the housing contains a pressurized gas compartment.
