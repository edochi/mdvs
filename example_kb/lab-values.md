# Prismatiq Lab Values

We are a small group. We move fast, but we don't cut corners on reproducibility. Every experiment gets logged, every dataset gets versioned, every protocol gets reviewed. If you can't reproduce it, it didn't happen.

## Principles

1. **Document everything.** If it's not written down, it doesn't exist. Use frontmatter. Use tags. Future-you will thank present-you.
2. **Share early.** A half-finished draft shared with the group is worth more than a polished paper no one has seen.
3. **Break things carefully.** We work with expensive equipment and irreplaceable samples. Double-check before you run.
4. **Respect REMO.** He is a colleague, not a tool. Yes, procurement lists him under "Environmental Control Systems." We are working on it.
5. **Clean your bench.** This includes virtual benches. Archive what you're not using.
