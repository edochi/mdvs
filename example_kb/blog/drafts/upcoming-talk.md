# Preparing for Photonics Europe 2032

We got accepted for an oral presentation at Photonics Europe 2032! The talk will cover our full calibration pipeline — from REMO's environmental sensitivity discovery to the integrated drift correction model.

I'm planning to include live data from the sensor array if we can get the demo unit stable enough for travel. Marco thinks we can run the reconfiguration engine on a laptop if Chiara's wavelet preprocessing brings the latency down enough. Fingers crossed!
