# TODO-0085: Rework search command pipeline

## Summary

Rework the search command to use the pipeline model.

## Details

### Pipeline

```
read_config → read_index → load_model → embed_query → execute_search
```

### Structs

```rust
struct SearchProcessOutput {
    read_config: ProcessingStepResult<ReadConfigOutput>,
    read_index: ProcessingStepResult<ReadIndexOutput>,
    load_model: ProcessingStepResult<LoadModelOutput>,
    embed_query: ProcessingStepResult<EmbedQueryOutput>,
    execute_search: ProcessingStepResult<ExecuteSearchOutput>,
}

struct SearchCommandOutput {
    process: SearchProcessOutput,
    result: Option<SearchResult>,
}
```

### Pipeline control

- Model mismatch (config vs parquet metadata) → `Failed(UserError)` on `read_index`
- `--where` clause errors → `Failed` on `execute_search`
- `LoadModelOutput` and `run_load_model()` are shared with build
