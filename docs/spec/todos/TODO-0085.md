# TODO-0085: Rework search command pipeline

## Summary

Rework the search command to use the pipeline model.

## Resolution

Reworked `search::run()` to return `SearchCommandOutput` directly (no `anyhow::Result`). Pipeline is `read_config → read_index → load_model → embed_query → execute_search` (5 steps). Created 3 new shared step modules: `load_model` (loads embedding model from config), `embed` (embeds query string), `execute_search` (runs search via backend with `--where` quote validation). Pre-check errors (missing embedding_model, missing index, model mismatch) land on `load_model` step as `Failed(User)` to fail fast before expensive model loading. Removed `elapsed_ms` from `SearchResult`. Exit code 2 for failed steps.
