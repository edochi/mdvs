# TODO-0019: Global --verbose flag

## Summary

Add a `--verbose` / `-v` global CLI flag that enables detailed progress output across all commands.

## Details

- Global flag on the root clap command, same pattern as `--output`
- When enabled, commands print additional detail to stderr (e.g., per-file status during build, per-field info during check, timing)
- Default behavior (no flag) stays concise as it is today
- Particularly useful for incremental build (showing which files were unchanged/changed/new/deleted)
