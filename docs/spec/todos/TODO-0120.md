# TODO-0120: Step tree: core types — Step, StepOutcome, StepError

## Summary

Create the foundational generic `Step<O>` type and its supporting types. This is the first piece of the TODO-0119 architecture — everything else builds on it.

## Details

Create `src/step.rs` with:

### Types

- `Step<O>` — generic tree node with `substeps: Vec<Step<O>>` and `outcome: StepOutcome<O>`
- `StepOutcome<O>` — enum: `Complete { result: Result<O, StepError>, elapsed_ms: u64 }` | `Skipped`
- `StepError` — struct: `kind: ErrorKind` + `message: String`
- `ErrorKind` — enum: `User` | `Application`
- Type aliases: `type FullStep = Step<Outcome>`, `type CompactStep = Step<CompactOutcome>`

### Methods

- `StepOutcome::elapsed_ms(&self) -> Option<u64>` — accessor
- `Step<Outcome>::to_compact(&self) -> Step<CompactOutcome>` — recursive conversion (calls `outcome.to_compact(&self.substeps)`)

### Free functions

- `has_failed(step: &Step<Outcome>) -> bool` — recursive, checks for `Err` in any node
- `has_violations(step: &Step<Outcome>) -> bool` — recursive, delegates to `Outcome::contains_violations()`

### Test helpers (`#[cfg(test)]`)

- `Step<Outcome>::unwrap_build() -> &BuildOutcome`
- `Step<Outcome>::unwrap_search() -> &SearchOutcome`
- `Step<Outcome>::unwrap_check() -> &CheckOutcome`
- `Step<Outcome>::unwrap_init() -> &InitOutcome`
- `Step<Outcome>::unwrap_update() -> &UpdateOutcome`
- `Step<Outcome>::unwrap_info() -> &InfoOutcome`
- `Step<Outcome>::unwrap_clean() -> &CleanOutcome`
- `Step<Outcome>::unwrap_error() -> &StepError`
- `Step<Outcome>::is_skipped() -> bool`
- One `unwrap_*` per leaf Outcome variant

### Notes

- `to_compact()` requires `Outcome` and `CompactOutcome` to exist (TODO-0122), but the method signature and stub can be written first.
- `StepError` must `derive(Clone)` for the `Err` branch of `to_compact()`.
- `Step<O>` does NOT derive `Serialize` — hand-impl comes in TODO-0124.
