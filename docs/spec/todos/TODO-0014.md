# TODO-0014: Build detects manual config changes via parquet metadata

## Summary

Build should compare mdvs.toml config against parquet metadata to detect manual edits (e.g. user changed model name directly in toml). Require `--force` to confirm, same as `--set-*` flags.

## Resolution

After filling missing config sections, build reads `BuildMetadata` from `files.parquet` and compares `EmbeddingModelConfig` and `ChunkingConfig` via `PartialEq`. Collects all mismatches into a single error message. Requires `--force` to proceed. First build (no parquets) never needs `--force`. Added `Clone` and `Eq` derives to config structs for composability.
