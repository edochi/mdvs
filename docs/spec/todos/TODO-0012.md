# TODO-0012: Store build metadata in parquet

## Summary

Write build configuration (model, revision, chunk_size, glob, built_at) as native parquet key-value metadata on chunks.parquet. Add a reader helper to extract it.

## Details

- Store metadata keys on the Arrow schema before writing chunks.parquet: `mdvs.model`, `mdvs.revision`, `mdvs.chunk_size`, `mdvs.glob`, `mdvs.built_at`
- Add a struct (e.g. `BuildMetadata`) to represent the stored metadata
- Add `read_build_metadata(path) -> Option<BuildMetadata>` helper in storage.rs
- `write_parquet` needs a way to accept custom key-value metadata (either a new function or extend the existing one)
- built_at should be ISO 8601 string in metadata for readability
