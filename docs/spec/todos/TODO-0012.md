# TODO-0012: Store build metadata in parquet

## Summary

Write build configuration (model, revision, chunk_size, glob, built_at) as native parquet key-value metadata on files.parquet. Add a reader helper to extract it.

## Resolution

Added `BuildMetadata` struct to `storage.rs` composing `EmbeddingModelConfig` and `ChunkingConfig`. Metadata stored as `mdvs.*` prefixed key-value pairs on the Arrow schema of `files.parquet` (the primary build manifest). Added `write_parquet_with_metadata()` and `read_build_metadata()` helpers. Build constructs and writes metadata on every build.
