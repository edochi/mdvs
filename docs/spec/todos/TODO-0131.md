# TODO-0131: Step tree: delete old pipeline, full cleanup

## Summary

Full cleanup: rewrite all 7 commands to call core functions directly (replacing `from_pipeline_result(run_*(...))` with `ScannedFiles::scan()` + timing + Step construction). Delete entire `src/pipeline/` directory. Remove all migration glue from `step.rs`. Clean up `output.rs`.

## Strategy

Each pipeline `run_*()` function wraps a core function with timing + `ProcessingStepResult`. Commands currently call `from_pipeline_result(run_*(), ...)` to convert. The cleanup eliminates both layers — commands call core functions directly and construct Steps inline.

A `Step::leaf()` constructor helper reduces the boilerplate of building leaf steps.

- `check::validate()` is already a core function in `src/cmd/check.rs` — `pipeline/validate.rs` is just a thin wrapper. Easy delete.
