# TODO-0010: Support length constraints on strings and arrays

## Summary

Add `min_length` and `max_length` constraints to String and Array fields for length validation during check.

## Details

**Toml representation:**
```toml
[[fields.field]]
name = "title"
type = "String"
min_length = 1
max_length = 100

[[fields.field]]
name = "tags"
min_length = 1    # at least one tag

[fields.field.type]
array = "String"
```

**Validation (check):** If `min_length`/`max_length` are set, check string character count or array element count. New ViolationKind or extend existing with detail.

**Inference:** Not inferred automatically — user-defined only.

**Storage/Query:** No change. Constraints are validation-only.

**Files:** `src/schema/config.rs` (add `min_length`/`max_length` to TomlField), `src/cmd/check.rs` (validate lengths)
