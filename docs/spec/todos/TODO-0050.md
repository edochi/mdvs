# TODO-0050: Fix String null serialization to Arrow NULL

## Details

In `storage.rs:174-182`, when a YAML frontmatter field contains `null` and the field is typed as String, `Value::Null.to_string()` produces the literal string `"null"` instead of Arrow NULL. Other types (Boolean, Integer, Float) correctly produce NULL for null values.

Fix: check for `Value::Null` before the match and return `None` (Arrow NULL).
