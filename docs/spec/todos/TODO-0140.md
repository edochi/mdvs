# TODO-0140: Global --dry-run flag

## Summary

Make `--dry-run` a global CLI flag (`#[arg(global = true)]`) available to all commands. Commands that write (init, update, build, clean) respect it. Read-only commands (check, search, info) ignore it silently.

## Current state

- `init` has `--dry-run` (per-command flag)
- `update` has `--dry-run` (per-command flag)
- `build` and `clean` don't have it

## Changes

1. Move `--dry-run` from init/update per-command args to the global `Cli` struct
2. Pass it down to all commands
3. Implement for `build`: validate + classify but skip embedding and index write
4. Implement for `clean`: show what would be deleted without deleting
5. Read-only commands: ignore the flag (no behavior change)
