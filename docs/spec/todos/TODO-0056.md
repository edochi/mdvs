# TODO-0056: Verify .mdvs/ is not a symlink before clean

## Details

`clean.rs` calls `remove_dir_all` on `.mdvs/` without checking if it's a symlink. An attacker could replace `.mdvs/` with a symlink to another directory, causing `clean` to delete unintended files. Should verify `.mdvs/` is a real directory and optionally check it contains expected files (e.g. `files.parquet`).

## Files

- `src/cmd/clean.rs` — add symlink check before deletion
