# TODO-0080: Shared step output structs and run functions

## Summary

Define the output structs and run functions for each processing step in `src/pipeline/` submodules. Each module owns its run function, output struct, and `StepOutput` impl.

## Resolution

All step modules implemented incrementally across TODO-0081 through TODO-0087 as each command was reworked. Each module provides a `run_*()` function that takes inputs, calls domain logic, measures elapsed time, and returns `ProcessingStepResult<T>`. Output structs implement `Serialize` and `StepOutput`. Steps are shared across commands as designed (e.g. `run_scan` used by check, build, init, update; `run_validate` used by check, build, init, update; etc.).
