# TODO-0003: Fix --auto-build flag on init

## Summary

Replace `--auto-build` (bool flag, always true) with `--suppress-auto-build` so users can disable auto-build. Default behavior remains auto-build enabled.

## Details

Currently `--auto-build` is defined with `default_value = "true"` which makes it a flag that's always true and cannot be set to false. `--auto-build=false` errors.

**Fix:** Replace with `--suppress-auto-build` flag (default absent = false). When present, auto-build is skipped. The `auto_build` parameter passed to `init::run()` becomes `!suppress_auto_build`.

**Files:** `src/main.rs` (CLI definition)
