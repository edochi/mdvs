# TODO-0003: Fix --auto-build flag on init

## Summary

Replace `--auto-build` (bool flag, always true) with `--suppress-auto-build` so users can disable auto-build. Default behavior remains auto-build enabled.

## Resolution

Replaced `--auto-build` (with `default_value = "true"`) with `--suppress-auto-build` boolean flag. Default absent = auto-build enabled. When present, `!suppress_auto_build` is passed to `init::run()`.
