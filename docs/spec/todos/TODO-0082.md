# TODO-0082: Rework build command pipeline

## Summary

Rework the build command to use the pipeline model. Build is the most complex pipeline (7 steps).

## Details

### Pipeline

```
read_config → scan → validate → classify → load_model → embed → write_index
```

### Structs

```rust
struct BuildProcessOutput {
    read_config: ProcessingStepResult<ReadConfigOutput>,
    scan: ProcessingStepResult<ScanOutput>,
    validate: ProcessingStepResult<ValidateOutput>,
    classify: ProcessingStepResult<ClassifyOutput>,
    load_model: ProcessingStepResult<LoadModelOutput>,
    embed: ProcessingStepResult<EmbedFilesOutput>,
    write_index: ProcessingStepResult<WriteIndexOutput>,
}

struct BuildCommandOutput {
    process: BuildProcessOutput,
    result: Option<BuildResult>,
}
```

### Pipeline control

- If `validate` completes with violations → remaining steps are `Skipped`, result is `None`
- If `classify` shows 0 new/edited → `load_model` and `embed` are `Skipped` (nothing to embed)
- If `load_model` fails → `embed` and `write_index` are `Skipped`
- Replaces the current `BuildOutcome` enum (TODO-0073)
