# TODO-0004: Rename --where-clause to --where

## Summary

Rename the search `--where-clause` flag to `--where` to match the spec.

## Resolution

Changed `#[arg(long, name = "where")]` to `#[arg(long = "where")]` in the Search command definition. The `name` attribute only sets the value placeholder, not the flag name — `long = "where"` is what controls the CLI flag.
