# Recipes

Walkthroughs for pointing mdvs at common markdown ecosystems and the agents that work in them.

- **[Agent harnesses](./recipes/agent-harnesses.md)** — Wiring mdvs into Claude Code, Codex, Cursor, OpenCode, Antigravity. Skill file, project-rules snippet, validate-on-write hook, search-nudge hook. Per-platform pages for copy-paste install; overview page for the architecture and how to extend to a new harness.
- **[Obsidian](./recipes/obsidian.md)** — YAML-frontmatter vaults, `.mdvsignore` patterns, Dataview caveats, common validation setups
- **[Hugo](./recipes/hugo.md)** — Mixed-format sites (YAML / TOML / JSON), native TOML date queries, forced-format mode for opinionated repos
- **[CI](./recipes/ci.md)** — Running `mdvs check` in a pipeline as a frontmatter linter
